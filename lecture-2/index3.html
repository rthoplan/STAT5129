Is Web Scraping Only for Structured Data?

Short answer: No. Explore the different types of data you can extract.

Examples: Tables & Lists

Data in `<table>`, `<ol>`, or `<ul>` tags. The HTML structure directly maps to a data table, making it easy to scrape.

Example: Free Text

Information is in paragraphs. You scrape the raw text, then use tools like regex, string rules, or NLP to pull out specific facts (dates, names, prices).

Example: Non-HTML Files

Data is inside an image or PDF. This requires an OCR (Optical Character Recognition) tool to convert the image to text. The results can be **noisy** and often need cleaning.
