Your Ethical Checklist

Accessing web data responsibly involves more than just checking `robots.txt`. Here is a simple checklist to follow:

  1. Check `robots.txt` first: Respect the site's crawling rules.
  2. Read the Terms of Use: Look for rules on automated access or data reuse.
  3. Check for Copyright/License: Does the site have a Creative Commons or other license?
  4. Look for an API: Always prefer using an official API if one is available.
  5. When in doubt, ask: If you're unsure, contact the site owner for permission.
